# MovieHeavens
### 基于Pyqt4的电影天堂电影搜索工具

### 关于开发
最初为了避免找电影期间的各种广告,以及各种页面跳转

### 后续更新

使用过程中如果发现无法检索到结果,应该是网站发生了更新

1. 重新更新了电影天堂的搜索
2. 将 python 由 2 切换到 3


### 使用

命令行下

```python
python3 movies.py
```
